# Extreme Roles
AmongUsの上級者(初心者)向けのロール追加MOD、主な特徴として以下があります<br>
* 第三陣営「ニュートラル」の追加
* AmongUs公式役職とMODの役職の併用可能
* すべてのMOD役職が複数人アサイン可能
* すべての役職に対して、細かい設定(視界、視界効果、キルクールなど)が可能
* 多彩なゲームオプション(シャッフルの強度、エンジニアのベント調整など)
* MODオプションのインポート/エクスポート機能
* MODの更新/バージョンチェック機能

このMODはAmong UsまたはInnersloth LLCとは提携しておらず、含まれるコンテンツはAmong UsやInnersloth LLCから支援、援助されているものではありません。

## 現在実装検討中の役職及び機能
- ボマー(インポスター)
   - プレイヤーを爆弾にする(キスミィさん提案)
   - 次期リリースバージョン(v1.18.0.0)で実装予定
   - 現在(2022/02/16)開発中のバージョンで実装完了
- エージェンシー(役職名カラー未定、クルー)
   - プレイヤーのタスクを肩代わりする(なりすさん提案)
   - 次期リリースバージョン(v1.18.0.0)で実装予定
   - 現在(2022/02/18)開発中のバージョンで実装完了
- ???(第三陣営)
   - マップ内の指定箇所に魔法陣を貼ることで勝利する。魔法陣内のプレイヤーの移動速度が減少する
   - 次期リリースバージョン(v1.18.0.0)に実装できたら良いなぁ(希望的観測)
   - 現在(2022/02/16)役職名とAmongUsとの兼ね合い、仕様検討中
- メリー(インポスター)
   - マップ内にインポスターだけ見えて使用できるベントを設置する(マーリンは見える、ねこわささん提案)
   - ベントは設置されただけでは有効化されず、近くにインポスター役職以外が近づいた時に有効化される
   - 現在(2022/02/18)役職名と仕様検討中
