# Extreme Roles
AmongUsの上級者(初心者)向けのロール追加MOD、主な特徴として以下があります<br>
* 第三陣営「ニュートラル」の追加
* AmongUs公式役職とMODの役職の併用可能
* すべてのMOD役職が複数人アサイン可能
* すべての役職に対して、細かい設定(視界、視界効果、キルクールなど)が可能
* 多彩なゲームオプション(シャッフルの強度、エンジニアのベント調整など)
* MODオプションのインポート/エクスポート機能
* MODの更新/バージョンチェック機能

このMODはAmong UsまたはInnersloth LLCとは提携しておらず、含まれるコンテンツはAmong UsやInnersloth LLCから支援、援助されているものではありません。
ここに含まれる素材の一部は、Innersloth LLCの所有物です。©Innersloth LLC.

## 現在確認されている不具合
- ラバーズ等で「別役職の能力を持つ」がオンの時、正しく別役職の能力が付与されない問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、別役職に能力持ちの役職をアサインしようとするとnull例外アクセスエラーが発生する問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、別役職のキルチェックロジック(タイムマスターのタイムシールド等)が正しく動かない問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、別役職の死亡時のアクション(アリスの勝利判定等)が正しく動かない問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、別役職の追放時のアクション(ジェスターの勝利判定等)が正しく動かない問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、別役職の死亡にホックしているアクション(ウィスパーの耳すまし等)が正しく動かない問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、別役職のリセット処理が正しく動作しない問題(現在開発中のバージョンで修正済み)
  - リセット処理が正しく動かないため、恐らくラバーズ等で「別役職の能力を持つ」がオンの時にその役職を正しくサイドキックにすることが出来ないはず(正しくサイドキックにできてることを確認)
- ラバーズ等で「別役職の能力を持つ」がオンの時、ゲーム終了時に表示される役職タグで別役職のタグが表示されない問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、ゲームの役職説明の色がおかしい問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンの時、ゲームの役職説明が一部未翻訳の問題(現在開発中のバージョンで修正済み)
- ラバーズ等で「別役職の能力を持つ」がオンで、役職がアサインされてない時、役職の情報が正しく表示されない問題(現在開発中のバージョンで修正済み)

## 現在実装検討中の役職及び機能
- Extreme Rolesのアドオン「Extreme Skins」
   - 誰でも簡単にスキンを作成できることを目標に作成しています
- ヤンデレ(第三陣営、シルシラさん提案、バイオレットレッド)
   - ベント使用可能
   - ゲーム開始時に想い人が決定される
   - 想い人の付近にいる人(設定時間と距離は変更可能)は「邪魔者」として登録され、キルをすることが可能になる(「邪魔者」の登録上限数もある)
   - 会議と会議との間が一定時間以上あり一回も「邪魔者」が作成されていない場合、その会議の後次の会議までキル能力を得る
   - 最初のキルクールは役職にセットされているもの
   - 役職の人がそれをキルすることで、次回のキルクールタイムを大幅に減らす(80％位、調整可)
   - 「邪魔者」以外をキルした場合、次回のキルクールタイムが大幅に増える(デフォルトから1.25倍(調整可)され続ける、会議でリセット)
   - 「邪魔者」が居なくなった場合、キル能力を失う
   - 想い人が居なくなった場合、常時キル出来るようになる。ただし、全て「邪魔者」以外をキルした扱いになる
   - 想い人及び「邪魔者」にはターゲット矢印が表示される(オンオフ可)
   - 勝利条件1：自分が生存しており想い人、以外の生存者が無害かつ2人以下(この場合、想い人は勝利扱いになる)
   - 勝利条件2：自分以外全てを殺害する(この場合、想い人は勝利扱いにならない)
